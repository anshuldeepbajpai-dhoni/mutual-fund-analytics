# Mutual Fund Analytics Platform

## Project Overview

This project focuses on analyzing mutual fund data using Python, SQL, and data visualization techniques. The objective is to build a complete analytics pipeline for mutual fund performance, NAV trends, investor transactions, AUM analysis, and business insights.

---

## Tech Stack

* Python
* Pandas
* NumPy
* SQLite
* SQLAlchemy
* Jupyter Notebook
* Git & GitHub
* Power BI (Upcoming)
